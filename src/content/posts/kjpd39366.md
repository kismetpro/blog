## GitHub 集成 Claude 与 Codex AI 编程代理，支持多模型协同开发

GitHub 于 2 月 4 日宣布，正式将 Anthropic 的 Claude 和 OpenAI 的 Codex AI 编程代理集成至 GitHub 平台。该功能目前处于公开预览阶段，Copilot Pro Plus 及 Copilot Enterprise 订阅用户可在 GitHub 网页端、移动端及 Visual Studio Code 中直接调用。
作为 Agent HQ 愿景的核心组成部分，开发者现可针对特定任务、Issue 或 Pull Request 选择不同的 AI 代理。GitHub 首席产品官 Mario Rodriguez 表示，此举旨在消除开发过程中的工具切换摩擦。未来，GitHub 计划引入 Google、Cognition 及 xAI 的相关代理。
