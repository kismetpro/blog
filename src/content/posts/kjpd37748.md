## OpenAI 披露 Mixpanel 安全事件，部分 API 用户分析数据或受影响

OpenAI 通报称，第三方数据分析服务商 Mixpanel 于 2025 年 11 月 9 日遭遇未授权访问，攻击者导出了一份包含部分 API 用户相关分析信息的数据集。事件未影响 OpenAI 自身系统，也未涉及聊天内容、API 请求、API 使用量、密码、凭证、API 密钥或支付信息。
受影响的数据可能包括 API 账户姓名、邮箱、粗略地理位置、操作系统与浏览器类型、引荐来源，以及与账户关联的组织或用户 ID。OpenAI 表示已将 Mixpanel 从生产环境移除，正在通知相关用户，并持续监测潜在滥用风险。公司同时开展更广泛的供应商安全审查，并提醒用户警惕钓鱼与社工攻击。
