## Microsoft 停止 Windows 等产品的电话激活方式，全面转向线上激活

Microsoft 已停止 Windows 及所有软件产品的电话激活，用户拨打原有号码将被指引前往官网完成在线激活。该变化自本月3日开始，此前 Windows 11 仍支持电话方式激活。
电话激活自 Windows XP 引入，曾为无稳定网络用户提供替代方案。Microsoft 自动语音信息确认，今后仅支持基于互联网的激活。
