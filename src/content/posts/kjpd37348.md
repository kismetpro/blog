## 苹果推出小程序合作伙伴计划，佣金降至 15%

苹果宣布推出小程序合作伙伴计划（Mini Apps Partner Program），参与计划的开发者在销售符合条件的App内购买项目时，可享受 15% 的收益抽成减免（一般情况为抽佣 30%）。
要获得该计划资格，主应用必须在 iOS 或 iPadOS 上的 App Store 中可用，需要支持高级商务 API 和声明年龄评级 API 等特定苹果技术，并使用苹果的应用内购买系统。
