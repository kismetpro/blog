## Microsoft 推出 Excel Agent Mode 全面上线，带来多步智能工作流

Microsoft 宣布 Excel 的 Agent Mode 已在 Excel for Web 正式上线，面向拥有商业版 Microsoft 365 Copilot 或 Microsoft 365 Premium 订阅的用户开放。该模式从基础助手升级为可在工作簿中直接构建多步工作流的智能代理体验，可支持预算分析、数据建模及动态图表生成等复杂任务。
该功能利用最新的 OpenAI 模型实现实时推理与结果解释，用户可在工作簿侧栏中查看其执行过程与验证结果。欧洲地区暂未开放该功能。
