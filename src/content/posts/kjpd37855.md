## Moonshot AI 推出 K2 Vendor Verifier 监测各厂商 Kimi K2 API 精度

Moonshot AI 于 2025 年 11 月发布开源工具 K2 Vendor Verifier，用于评估不同厂商部署的 Kimi K2 模型在工具调用 (toolcall) 环节的精度与一致性。该工具比较官方接口与各供应商接口的触发相似度及 JSON 架构准确率，结果显示 Moonshot AI、Fireworks 等平台精度达 100%，而部分通过 OpenRouter 提供的模型准确率低至约 72%。
K2 Vendor Verifier 目前支持 K2-thinking 与 K2-0905 两个版本的评测，可帮助开发者发现模型调用差异并优化版本设置。Moonshot AI 称，该机制将用于长期监测各 API 表现，确保用户获得稳定一致的 Kimi K2 使用体验。
