## 📱

GitHub 在遭到强烈反对后推迟对 GitHub Actions 收费的计划
GitHub 表示已推迟自托管 GitHub Actions 的计费变更以便重新评估方案，同时承诺于 2026 年 1 月 1 日继续把托管 runner 价格最多下调 39%。公司承认此次决策未充分包含用户意见，现将广泛听取开发者、客户和合作伙伴的反馈，利用这些意见调整 Actions 路线图，并通过持续交付努力重建信任与合作关系。
