## 微软内部大规模推广 Claude Code，鼓励非技术员工使用 AI 编程

微软正在其最重要的工程团队中广泛推广 Anthropic 的 Claude Code，甚至鼓励没有编程经验的员工使用该工具进行原型设计。据消息人士透露，微软 CoreAI 团队和负责 Windows、Microsoft 365、Outlook 等产品的体验与设备部门都被要求安装 Claude Code。
微软软件工程师现在需要同时使用 Claude Code 和 GitHub Copilot 并提供对比反馈。尽管微软向客户销售 GitHub Copilot 作为首选 AI 编程工具，但公司内部越来越倾向于使用 Claude Code。微软已成为 Anthropic 的顶级客户之一，并将销售 Anthropic AI 模型计入 Azure 销售配额。微软与 Anthropic 在 11 月签署协议，允许 Microsoft Foundry 客户访问 Claude 系列模型，Anthropic 承诺购买 300 亿美元的 Azure 计算容量。
