## 任天堂向 GitHub 多个 Switch 模拟器项目发出 DMCA 通知

任天堂代表于 2 月 13 日向 GitHub 上所有任天堂 Switch 模拟器及其分支项目发出 DMCA 通知。报道提及 Eden、Citron、Kenji-NX、MeloNX 等项目及 Sudachi、Skyline 等已不活跃项目。发稿时相关仓库仍可访问但预计将被移除
报道称部分仍在开发的项目已在 GitHub 以外托管或备份仓库。任天堂主张相关项目涉及规避技术保护措施并称 Switch 游戏仅能在官方硬件上合法游玩
