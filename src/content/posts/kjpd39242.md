## 微软 PowerToys 拟为 Windows 11 引入顶部菜单栏功能

微软 PowerToys 团队正计划为 Windows 11 开发名为 “Command Palette Dock” 的顶部菜单栏。该可选 UI 旨在提供工具快速访问及系统资源监控功能，支持高度自定义位置与外观。作为去年推出的 Command Palette 启动器的补充，该功能目前处于反馈征集阶段。开发者现可通过 GitHub 仓库在 Visual Studio 中导入并体验早期版本。
