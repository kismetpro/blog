## Cloudflare 推出 Markdown for Agents 功能，支持 HTML 实时转换为 Markdown

Cloudflare 于 2026 年 2 月 12 日推出 Markdown for Agents 功能，支持在边缘网络将 HTML 网页实时转换为 Markdown 格式。AI 代理及爬虫通过在请求头中添加 Accept: text/markdown 即可直接获取结构化数据。
该功能旨在优化 AI 处理效率并降低 Token 成本。数据显示，将 HTML 转换为 Markdown 可减少约 80% 的 Token 消耗。系统同时提供 x-markdown-tokens 响应头以反馈 Token 估算值。目前该功能已开启 Beta 测试，Pro、Business 及 Enterprise 计划用户可免费启用。
