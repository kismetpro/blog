## 月之暗面完成 5 亿美元 C 轮融资，全球付费用户数月增速 170%

AI 公司月之暗面完成 5 亿美元 C 轮融资，创始人杨植麟在内部信中透露，公司全球付费用户数月增速达 170%。11 月以来，受 K2 Thinking 大模型带动，月之暗面海外 API 收入增长 4 倍。
今年 9 月，月之暗面推出 Agent 功能 OK Computer，可调用虚拟电脑工具进行网站开发、数据分析等操作。基于此功能，公司启动会员制收费，国内分为每月 49 元、99 元、199 元三档。杨植麟表示，C 轮资金将用于扩增显卡，加速 K3 模型训练研发。
