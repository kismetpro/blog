## 特斯拉拟于 2026 年第二季度停产 Model S 与 Model X

埃隆·马斯克（Elon Musk）在 1 月 28 日的财报电话会议上宣布，特斯拉计划于 2026 年第二季度停止生产 Model S 和 Model X 车型。此举旨在为弗里蒙特（Fremont）工厂生产 Optimus 人形机器人腾出空间。马斯克表示，特斯拉正全面转向以自动驾驶为核心的业务模式。
Model S 与 Model X 分别于 2012 年和 2015 年上市，近期销量持续低迷。2025 年，包含上述车型在内的类别销量同比下降 40.2%。目前特斯拉正处于转型期，其 2025 年第四季度利润同比大幅下降 61%。
