## X 推出按使用量付费 API 模型 beta 吸引开发者

X 宣布，其开发者平台开启“按使用付费”API 模型的 beta 测试，让开发者仅为实际调用量付费。参与该测试的开发者将获得约 500 美元的使用券，用于激励早期接入。此前最低月付费 200 美元。
此次调整降低 了 API 接入门槛，改善此前因高额订阅费挡住小型开发者和创新工具使用者的局面。
