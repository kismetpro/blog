## GitHub 今日遭遇短暂宕机，现已修复

北京时间 11 月 19 日 04:39，GitHub出现全球用户的服务故障，导致通过 SSH 和 HTTP 的 Git 操作（推送、克隆等）无法正常进行，同时部分 Codespaces 也受到影响。北京时间 6:00 左右，GitHub修复该故障。
