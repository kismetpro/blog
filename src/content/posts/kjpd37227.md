## 微软在 Bing 搜索 Chrome 时 “ 贿赂 ” 用户使用 Edge 浏览器

当用户在 Bing 上搜索"Chrome"时，微软现在会显示广告，提供 1,300 个 Microsoft Rewards 积分来鼓励用户继续使用 Edge 浏览器而不是下载 Chrome。这些积分可以兑换亚马逊礼品卡、Roblox 游戏币或 Spotify Premium 三个月会员等奖励，也可以捐赠给非营利组织。
微软在广告中声称 Edge 与 Chrome 使用相同的 Chromium 技术，并列出了 Edge 的优势功能，包括 VPN 服务和 AI 个性化主题生成。这类针对 Chrome 的推广广告在 Bing 上大约每两周更新一次，且无法被用户关闭。

*Windows Latest*
