## 近 90% 的 Windows 游戏现已可在 Linux 上运行

最新数据显示，通过 Proton/Wine 等兼容层，近 90% 的 Windows 游戏现在可以在 Linux 系统上运行， Linux 作为游戏平台的可行性达到了历史新高。“可运行”并不代表“无障碍体验”——部分游戏在 Linux 上可能仍受反作弊机制、优化程度或硬件支持限制影响。
