## 苹果强制 GitHub 删除 8270 个泄露 App Store 代码仓库

苹果公司因配置错误意外泄露了新版 App Store 网页版的前端源代码后，多名开发者将完整源代码打包上传至 GitHub 仓库供用户下载研究。苹果迅速向 GitHub 发出 DMCA 删除通知，要求移除超过 100 个包含泄露源代码的仓库。
经过全面扫描，GitHub 发现总共 8270 个仓库包含苹果源代码，并按照 DMCA 要求全部删除。不过由于代码已被大量用户本地下载或镜像到其他平台，完全清除几乎不可能。考虑到泄露的前端代码并不构成真正的安全威胁，苹果的激进执法行为令人意外。
