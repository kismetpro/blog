## Google 首席工程师称 Claude Code 一小时完成其团队一年工作量

Google 首席工程师 Jaana Dogan 近日在社交平台 X 披露，AI 编程工具 Claude Code 仅用 1 小时便构建出其团队耗时一年开发的分布式代理编排器（distributed agent orchestrators）。Dogan 表示，由于大公司内部基础设施与外部脱节，开发效率受到限制。她建议开发者在专业领域尝试 AI 编程代理以验证其效能。
目前 AI 编程已在头部科技公司普及。Anthropic 首席执行官 Dario Amodei 称该公司 90% 的代码由 AI 编写，Google 和 Microsoft 此前披露的比例分别为 50% 和 30%。
