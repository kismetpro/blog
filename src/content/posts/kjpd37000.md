## 🤖

OpenAI 与亚马逊签署 380 亿美元云计算协议，亚马逊股价创历史新高
OpenAI 与 AWS 签订价值 380 亿美元的长期合作协议，OpenAI 将立即使用 AWS 的基础设施和英伟达 GPU 进行大规模人工智能模型训练与推理。
这标志着 OpenAI 在结束与微软的独家云合作后，进一步拓展其计算资源来源。亚马逊计划未来为其建设新的数据中心。受此消息推动，亚马逊股价周一上涨 4%，创下历史新高。
OpenAI 仍将维持与微软、谷歌、甲骨文等多家云服务商的合作，并持续扩大算力布局。业内认为，该举措有助于其为未来可能的首次公开募股（IPO）做准备。
