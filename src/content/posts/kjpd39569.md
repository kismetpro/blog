## 亚马逊限制内部使用 Claude Code 推广自研工具 Kiro

亚马逊（Amazon）近日发布内部指南，限制员工在生产环境中使用 Anthropic 的 AI 编程工具 Claude Code，转而强制推广自研工具 Kiro。尽管亚马逊是 Anthropic 的重要投资者，但内部规定要求使用 Claude Code 必须经过正式审批。此举引发约 1,500 名员工联名抗议，部分销售工程师质疑，公司在通过 Bedrock 平台向客户推销该工具的同时却限制内部使用。亚马逊官方回应称，目前已有 70% 的工程师使用 Kiro，暂无计划支持更多第三方开发工具。
