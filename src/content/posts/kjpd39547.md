## 微软拟为 Windows 11 默认只跑签名应用，并引入手机式权限弹窗

微软宣布调整 Windows 处理应用的方式，推出新的安全模型。公司正在开发 Baseline Security Mode，新装的 Windows 11 将默认只运行已正确签名的应用、服务、驱动，但用户和 IT 管理员仍可选择放行未签名程序。
同时将加入 User Transparency and Consent：应用请求文件、相机、麦克风等敏感资源时会弹窗提示，用户可逐个应用授权或拒绝，并随时修改。微软称这些改动会在不影响既有“表现良好”应用的前提下逐步落地。
