## Anthropic 发布 Claude Opus 4.5，相较于前代模型定价大幅降低

Anthropic 正式发布 Claude Opus 4.5 模型，该模型在编程、智能体和计算机使用方面表现卓越，被称为「世界上最好的模型」。新模型在软件工程基准测试中达到业界领先水平，在一项内部技术考试中甚至超越了所有人类候选者的表现。
Claude Opus 4.5 的定价大幅降低至每百万 token 5/25 美元，使 Opus 级别的能力更加普及。该模型在保持高质量输出的同时显著提升了效率，在某些任务中 token 使用量减少高达 65%。Anthropic 同时推出了新的「努力参数」功能，允许开发者在速度和能力之间进行权衡。此外，公司还更新了 Claude 开发者平台、Claude Code 以及消费者应用，新增了对 Excel、Chrome 和桌面端的支持。
