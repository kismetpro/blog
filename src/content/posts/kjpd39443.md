## X 推出 API 按量付费模式，单次调用 0.005 美元起

X 推出 API 按量付费（Pay-Per-Use）模式，旨在降低开发者准入门槛。新模式取消了此前高达 200 美元的固定月费，改为按请求计费：读取单条帖子需 0.005 美元，访问用户资料或发布帖子为 0.010 美元。原免费层级活跃用户将获 20 美元代金券用于过渡。
同时，X 引入了 xAI 积分激励机制，开发者可根据 X API 支出获得最高 20% 的 xAI API 积分返还。此外，X 还上线了全新开发者控制台、交互式 Playground 及 X 开发者套件（XDK），并提供管理控制面板（MCP）以优化资源监控。
