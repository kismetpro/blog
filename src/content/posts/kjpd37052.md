## Claude 推出限时 Code 额度赠送活动

Claude 近日推出限时促销活动，为 Pro 与 Max 个人订阅用户提供额外 Claude Code 使用额度。Pro 用户可获 250 美元额度，Max 用户可获 1000 美元额度，额度将按 API 定价标准自动应用于 Claude Code 网页端与移动端。
活动时间为 2025 年 11 月 4 日上午 9 时至 11 月 18 日晚 11 时 59 分（太平洋时间），额度将在期限结束时失效。活动仅面向个人 Pro 与 Max 用户，新用户可在促销期间订阅参与，名额有限。
