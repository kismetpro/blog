## 开发者怒批华为云 C++ SDK 质量：文档失效、基础代码错误频出

2 月 4 日，有开发者在华为云 C++ SDK 的 GitHub 项目中提交 issue，详细列举了该 SDK 存在的多项技术问题。该开发者控诉仓库 README 文档严重过时，SDK 文档中的 bash 注释使用了错误的
//
语法，示例代码存在未声明变量，且编译配置与文档不符。
此外，该开发者指出 SDK 代码也存在大量基础错误：函数声明缺失、正则表达式转义错误以及循环依赖处理不当等。该开发者表示，这些问题影响了 SDK 的正常编译和使用，并建议华为云团队改进代码质量和文档维护。
