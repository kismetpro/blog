## Gemini 负责人回应调整 Gemini API 免费额度：以保障高端模型算力供给，未来可能转向积分制

Google 开发者关系负责人 Logan Kilpatrick 在 Reddit 多次回应称，公司已下调或移除部分 Gemini 免费 API 的速率限制，以将算力集中投入 Gemini 3.0 Pro 与 Nano Banana Pro 等需求快速增长的模型。他表示，免费层主要用于快速测试模型功能，并非为长期项目提供稳定算力。未来可能会转向积分制度，注册赠送免费积分。
Kilpatrick 指出，近期免费层遭遇大规模滥用，也促使 Google 临时收紧额度，并强调付费用户不会受到影响。他称未来仍将保留免费层，但额度将随算力供给和新模型上线动态调整。
