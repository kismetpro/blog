## 火绒发布全新“强力卸载”Beta版工具

火绒安全团队发布面向互联网PC终端的全新“火绒强力卸载”Beta版软件，主打轻便、简单、专注，旨在解决传统卸载工具卸载不彻底、操作繁琐等问题，为用户提供更彻底的软件移除能力和更纯净的系统环境。据官方介绍，该工具体积小、资源占用低，支持 Windows 7、8/8.1、10、11 及 Windows Server 2008 R2 及以上版本，可实时记录软件安装行为，并在卸载时精准清除文件、注册表项及关联组件。
官方提示：当前版本删除文件操作可能存在风险，建议优先在虚拟环境中试用。
