## JavaScript 诞生 30 周年：10 天编程马拉松催生互联网基石

12 月 4 日是 JavaScript 编程语言诞生 30 周年。1995 年，网景公司工程师 Brendan Eich 在 10 天内完成了 JavaScript 的初始原型开发，这门语言如今运行在全球 98.9% 的网站上，成为互联网最重要的编程语言之一。
JavaScript 最初被命名为 Mocha，后改为 LiveScript，最终因网景与 Sun 公司的合作协议而定名为 JavaScript。尽管名称容易与 Java 混淆，但两者除语法相似外几乎毫无关联。根据 2024 年 Stack Overflow 开发者调查，JavaScript 连续 12 年成为使用率最高的编程语言，62% 的开发者在使用该语言。目前，由 Brendan Eich 等人发起的请愿书正要求甲骨文公司放弃 JavaScript 商标，让其成为公共使用的通用术语。
